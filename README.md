## 📷 多摄像头同步拍摄工具

本软件用于在同一台设备上**同时连接多个 USB 摄像头**，并支持**一键同步拍摄**和保存图片。适用于驾驶行为采集、多视角图像采集、监控记录等场景。

### ✨ 功能特点

* 支持 **3 个摄像头** 同时连接（旧版本支持 2 个摄像头）
* 点击一次按钮即可 **同步拍摄 3 张图像**
* 自动以时间戳命名并保存图片，方便管理
* 支持选择保存目录
* 界面简单，操作方便

### 🖥️ 使用方法

1. 将摄像头连接到电脑 USB 接口
2. 运行软件
3. 确保软件界面中已经识别到对应数量的摄像头
4. 点击 **“拍摄”** 按钮
5. 程序会自动保存 3 张同步照片至选择的文件夹中

### 📁 图片命名方式

保存的图片将按以下格式命名：

```
拍摄区域名称_frame_HH_MM_SS_摄像头编号.jpg
```

拍摄区域名称：`double`，`inner_bullseye`，`inner_single`，`miss`，`mix`，`mix_inner_outer`，`outer_bullseye`，`outer_single`，`triple`.
摄像头编号：0，1，2
